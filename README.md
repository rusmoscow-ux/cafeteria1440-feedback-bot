# Cafeteria 1440 Feedback Bot

Telegram-бот для обратной связи по кафетерию 1440.

## Сценарии

Пользователь выбирает один из вариантов:

- 👍 Сегодня понравилось
- ✨ Хотелось бы добавить
- 🔧 Стоит поправить
- 💬 Просто комментарий

После комментария бот предлагает фото только для сценариев, где оно может быть полезно:

- 👍 Сегодня понравилось
- 🔧 Стоит поправить

Для сценария «Хотелось бы добавить» фото не запрашивается, чтобы не усложнять путь пользователя.

## Отчеты

Бот сохраняет все комментарии в SQLite и умеет делать AI-сводку:

- автоматически каждый день в `DAILY_REPORT_TIME`;
- вручную командой `/daily_report` в группе.

## Railway variables

Нужны переменные:

- `BOT_TOKEN` — токен нового Telegram-бота;
- `MANAGER_CHAT_ID` — ID группы, куда отправлять комментарии;
- `OPENAI_API_KEY` — ключ OpenAI API;
- `TIMEZONE` — например `Europe/Moscow`;
- `DAILY_REPORT_TIME` — например `20:00`.

Дополнительно можно задать:

- `OPENAI_MODEL` — по умолчанию `gpt-4o-mini`;
- `FEEDBACK_DB_PATH` — по умолчанию `feedback.db`.

## Команды

- `/start` — оставить комментарий;
- `/id` — получить ID текущего чата;
- `/daily_report` — вручную сформировать сводку за сегодня.

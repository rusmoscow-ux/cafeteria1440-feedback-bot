# Cafeteria 1440 Feedback Bot v3

Telegram-бот обратной связи для кафетерия 1440.

## Что делает бот

- Собирает комментарии по 4 сценариям:
  - 👍 Сегодня понравилось
  - ✨ Хотелось бы добавить
  - 🔧 Стоит поправить
  - 💬 Просто комментарий
- После каждого комментария предлагает добавить фото.
- Отправляет комментарий и фото в управленческий чат.
- Сохраняет все отзывы в SQLite.
- Делает AI-сводку по команде `/daily_report`.
- Автоматически отправляет ежедневный отчет в `DAILY_REPORT_TIME`, по умолчанию в 20:00.

## Что исправлено в v3

- Ручная команда `/daily_report` не обнуляет и не закрывает дневной отчет.
- Автоматический отчет в 20:00 теперь строится по времени получения отзыва ботом (`created_at`), а не по отображаемой дате.
- Если в день нет отзывов, бот всё равно отправит сообщение «За сегодня отзывов не было», чтобы было понятно, что планировщик сработал.
- Фото можно добавить к каждому из 4 блоков.

## Railway variables

Обязательные:

- `BOT_TOKEN`
- `MANAGER_CHAT_ID`
- `OPENAI_API_KEY`

Опциональные:

- `TIMEZONE` — по умолчанию `Europe/Moscow`
- `DAILY_REPORT_TIME` — по умолчанию `20:00`
- `OPENAI_MODEL` — по умолчанию `gpt-4o-mini`
- `FEEDBACK_DB_PATH` — по умолчанию `feedback.db`

## Проверка

1. Оставьте 1–2 тестовых отзыва.
2. В управленческом чате выполните `/daily_report`.
3. В 20:00 бот должен автоматически прислать отчет за день.

Важно: если Railway пересоздает контейнер без постоянного диска, SQLite-файл может потеряться после деплоя. Для долгосрочного хранения лучше подключить Railway Volume или PostgreSQL.
